兔年大吉，开工大吉

呢喃细语

开篇

2023年2月1日，新的一个工作年度算是完完全全的开始了。

过去的2022年，发生了许多事情，但是2022年度本身，可以说是彻彻底底的结束了。

新的年度既已来到，那就要定下目标，严格执行。

目标

1、工作

找到一份工作，年薪在12W-20W之间，岗位以技术岗为主，程序员或技术经理均可。

2、技能提升

2.1 开发能力

利用B站及各种网络资源，以自学方式，提升以下几种开发语言的开发能力：

  • C++\QT

  • Java\SpringBoot

  • Vue\Openlayer\leaflet

  • Python

2.2 项目管理能力

利用B站及各种网络资源，以自学方式，获取以下几种资格证书：

  • 信息系统项目管理师
    （切记：时刻关注报名登记消息，不要忘了）
  • 注册测绘师

2.3 驾照

争取上半年通过驾考剩余科目考试，拿到驾照，下半年通过租车方式提升车感及熟练度，能在来年春节期间驾车拜亲访友、郊游出行。

3、锻炼身体

参照抖音各类减肥健身视频，工作日以室内锻炼为主，周末外出有氧运动为主，减去大肚囊。
